## Фаза 2: Опциональные компоненты T3

> T3 (Со-мыслитель) включает базовый набор из коробки. Дополнительные агенты — опциональные компоненты, добавляются когда появляется потребность.
> Единственная пользовательская классификация — T1-T4 (DP.ARCH.002). Отдельной оси «уровней» нет.

| Компонент | Что включает | Когда добавлять | Как добавить |
|-----------|-------------|-----------------|--------------|
| **Базовый T3** (текущий шаблон) | Claude Code + Стратег + governance hub (DS-strategy) | Сразу (fork + setup.sh) | Уже в шаблоне |
| **+ Экстрактор** | Knowledge Extraction | Когда пользователь создаёт первый Pack | Пользователь сам клонирует DS-extractor-agent + настраивает промпты |
| **+ Синхронизатор** | Cross-repo sync, code-scan | Когда ≥3 downstream-репо | Пользователь сам клонирует DS-synchronizer + настраивает scheduler |

### Почему не всё сразу

- **Экстрактор** требует Pack (source-of-truth) — без него некуда извлекать знания
- **Синхронизатор** требует ≥3 репо — иначе нечего синхронизировать
- Принцип: **минимальная сложность на старте**, градиентное усложнение по мере роста
